# Planning Notes

This directory contains focused planning notes, investigations, and contracts that are too detailed
for `ROADMAP.md`.

Use this area for:

- FABLE-C workbook structure investigations;
- scenario selection and scenario definition contracts;
- output table and headline indicator curation notes;
- notebook/user-guide design notes;
- benchmark validation protocols;
- sanitized findings from public or approved workbook inspections.

Do not store original FABLE-C workbook binaries, generated Python models, raw extracts, local logs,
private transcripts, or full validation reports here. Keep those under ignored `tmp/` paths unless
the maintainer explicitly approves a small sanitized tracked artifact.
